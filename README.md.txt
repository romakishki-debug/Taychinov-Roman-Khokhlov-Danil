Описание проекта
Проект посвящён реализации базовых алгоритмов машинного обучения:
* Линейная регрессия (реализация с нуля)
* Логистическая регрессия (реализация с нуля)
* Классификация с использованием дополнительной модели (**Decision Tree**)

Основной акцент сделан на реализации градиентного спуска вручную, анализе качества моделей и визуализации процесса обучения. Проект выполнен в Google Colab.

Структура проекта
project/
├── notebook.ipynb
├── README.md
├── requirements.txt
├── data/
│   └── a_steam_data_2021_2025.csv
└── report.pdf
 
Данные
* Источник: публичный датасет Steam (2021–2025)
* Регрессия: предсказание `price`
* Классификация: `is_popular` (rating ≥ 80)

Реализованные модели

Линейная регрессия

* Градиентный спуск
* MSE
* Коэффициенты и интерсепт
* График потерь
* Линия регрессии и доверительный интервал

Логистическая регрессия

* Сигмоида
* Log-loss
* Градиентный спуск
* Бинарная классификация

Метрики:
Accuracy, Precision, Recall, F1-score, ROC AUC, Confusion Matrix

Дополнительная модель

* Decision Tree (sklearn)
* Используется для сравнения качества классификации

Эксперименты

* Анализ влияния learning rate и количества эпох
* Сравнение моделей по accuracy и F1-score
* Интерпретация различий в качестве
